Overview

Project Summary

Motivation

Is estimated that foodborne diseases cause approximately 76 million illnesses,325,000 hospitalizations, and 5,000 deaths in the United States each year. The cause of foodborne illness include viruses, bacteria, parasites, toxins, and metals.

Research Question

How can the customers’ experiences help to identify possible foodborne diseases outbreaks over a geographical region?

Objectives

The primary objective for this project is to design a method of data mining that allows the making of a prediction based on a social network as Yelp by accessing user’s recommendations
The predictions we attempted are the following:

  • Identifying which symptoms go with which diseases to create training data for a Naive-Bayes algorithm.
  • Check an outside database for diseases that had outbreaks in each state.(CDC)
  • Predict the top possible diseases that might have an outbreak in each state based on Yelp reviews.

Hypothesis

Foodborne illness outbreak can be predicted using a social network.
